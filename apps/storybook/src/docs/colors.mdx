# Colors

Lumo theme is built on top of Tailwind CSS.

Since theme is extended, 
[all Tailwind default colors](https://tailwindcss.com/docs/colors) are 
available.

In addition to that, colors were tokenized to provide them meaning and 
semantics.

## Content colors

These colors are applicable for color utilities besides background.

export const mapCssVariableToPaletteItem = (cssVariable) => {
  let token = cssVariable.replace(/^--(?:color|background-color)-/, "")
  const isInverted = token.endsWith('-inv')
  if (isInverted) token = token.replace('-inv', '')

  const title = 
    token.at(0)?.toUpperCase() + 
    token.slice(1) + 
    (isInverted ? ' (inverted)' : '')
  const colors = [`var(${cssVariable})`]
  
  return <ColorItem title={title} colors={colors} />
}

<ColorPalette>
  {[
    "--color-primary",
    "--color-secondary",
    "--color-tertiary",
    "--color-primary-inv",
    "--color-secondary-inv",
    "--color-tertiary-inv",
  ].map(mapCssVariableToPaletteItem)}
</ColorPalette>

## Background colors

These colors are applicable only for background color utilities.

Such differentiation allows to have more control over color usage and creates
natural contrasting color pairs.
