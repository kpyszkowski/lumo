# Typography

Lumo theme is built on top of 
[Tailwind CSS](https://tailwindcss.com/docs/).

Predefined typography styles are 
[available unchanged](https://tailwindcss.com/docs/font-size).

Used typeface is [Manrope](https://fonts.google.com/specimen/Manrope/).
It is consumer app's responsibility to load the font. Preferably via [`next/font`](https://nextjs.org/docs/pages/getting-started/fonts)
or [`@fontsource-variable/manrope`](https://fontsource.org/fonts/manrope/install) accodringly to the project.
