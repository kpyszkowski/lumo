# Typography

Lumo theme is built on top of 
[Tailwind CSS](https://tailwindcss.com/docs/).

Predefined typography styles are 
[available unchanged](https://tailwindcss.com/docs/font-size).

Used typeface is [Satoshi](https://www.fontshare.com/fonts/satoshi).
It is consumer app's responsibility to load the font. UI library does not
include it for flexibility reasons.

export const fontSizes = [
  12, 14, 16, 18, 20, 24, 30, 36, 48, 60, 72, 96, 128
]

<Typeset 
  sampleText='The quick brown fox jumps over the lazy dog'
  fontFamily='"Satoshi Variable", sans-serif'
  fontSizes={fontSizes.map(size => `${size}px`)}
  fontWeight={400}
/>
